# GenMinds - Enterprise Intelligence Asset Management Platform

## Official Project Confirmation

This repository contains the **GenMinds** project - an Enterprise Intelligence Asset Management Platform.

### Project Identity
- **Project Name**: GenMinds
- **Full Name**: GenMinds - Enterprise Intelligence Asset Management Platform
- **Type**: Full-Stack Web Application
- **Purpose**: Enterprise platform for managing AI intelligence assets

### What is Included
This GenMinds repository contains:

1. **Frontend Application** (`GenMinds-B-Bm/` directory)
   - React 18 + TypeScript application
   - Intelligence asset management dashboard
   - Asset discovery and search interface
   - Benchmarking interface
   - Portfolio management UI

2. **Backend API** (`genminds-backend/` directory)
   - Node.js/Express server
   - REST API for asset operations
   - Mock data API for intelligence assets

3. **Deployment**
   - Google App Engine configuration
   - Production deployment at: https://trans-proposal-441718-c9.ew.r.appspot.com

### GenMinds Core Features
- **Asset Management**: Create, read, update, delete AI intelligence assets
- **Asset Discovery**: Search and browse intelligence assets
- **Benchmarking**: Evaluate AI model performance
- **Portfolios**: Organize assets into packages
- **Analytics**: Track asset usage and performance

### Technology Stack
- Frontend: React 18, TypeScript, Tailwind CSS, Vite
- Backend: Node.js, Express.js
- API: REST API
- Deployment: Google App Engine
- Database: Mock API data
